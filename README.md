# 🌐 Swastik Garg – Portfolio Website

A modern personal portfolio built using **Next.js, React, and Tailwind CSS** to showcase my projects, skills, certifications, and experience.

---

## 🚀 Live Website

👉 https://swastikengineer.vercel.app

---

## 📌 Features

* ⚡ Multi-page portfolio (Next.js App Router)
* 🎯 Clean and professional UI (Black theme)
* 📂 Separate pages for each section
* 🔗 External links for projects and certifications
* 📱 Fully responsive design
* 🚀 Deployed on Vercel
